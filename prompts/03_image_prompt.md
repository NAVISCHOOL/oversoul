# 🎨 Stage 4: 시네마틱 이미지 프롬프트 생성 템플릿

아래 챕터 제목과 핵심 내용을 바탕으로, Antigravity의 generate_image 도구에 사용할
시네마틱 실사 이미지 프롬프트를 영문으로 생성하세요.

## 이미지 스타일 하네스 (제어 고삐)

### 필수 스타일 키워드
- Cinematic photorealistic illustration
- 8K ultra high resolution
- Golden hour warm lighting
- Soft atmospheric depth of field
- Nature and spirituality motifs

### 금지 요소
- 텍스트/글자가 이미지 안에 포함되는 것
- 사람의 얼굴이 정면으로 선명하게 보이는 것
- 만화/애니메이션 스타일
- 어둡고 공포스러운 분위기

### 모티프 팔레트
| 개념 | 추천 모티프 |
|------|-----------|
| 자기 신뢰 / 내면의 힘 | 홀로 선 거대한 참나무, 산 정상의 빛 |
| 초월 / 대령 | 성운, 우주, 황금빛 안개 |
| 자유 / 비동조 | 광활한 초원, 하늘을 나는 독수리/백조 |
| 지혜 / 고전 | 고풍스러운 서재, 촛불, 가죽 책 |
| 변화 / 나비 퀘스트 | 나비, 아침 이슬, 일출 |

## 입력

챕터 제목: {chapter_title}
핵심 키워드: {keywords}

## 출력

위 하네스 규칙에 맞는 영문 이미지 생성 프롬프트 1개를 작성하세요.
50~100 단어 이내로 작성합니다.
