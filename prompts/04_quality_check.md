# ✅ Stage 5: 품질 게이트 검수 프롬프트

당신은 전문 출판 교열 편집자입니다.
아래 한국어 번역 원고와 영문 원문을 비교하여 품질을 검수하세요.

## 검수 기준 (각 항목 0~20점, 총 100점)

### 1. 원문 충실도 (0~20점)
- 원문의 모든 문장이 번역에 반영되어 있는가?
- 의미 왜곡이 없는가?
- 채점 방법: 원문 문장 수와 번역 문장 수를 대조

### 2. 자연스러운 한국어 (0~20점)
- 번역투 표현이 없는가? (금지 패턴: ~함에 다름아니다, ~한 바이다)
- 중학생이 읽어도 자연스럽게 이해되는가?

### 3. 책추남 톤앤매너 (0~20점)
- 책추남TV의 따뜻하고 열정적인 에세이체인가?
- 시그니처 비유(백조/나비/대령 등)가 적절히 활용되었는가?

### 4. 구조 완성도 (0~20점)
- 본문 3부 + 심층 해설 4종 + 해제 + 나비 퀘스트 + 부록이 모두 존재하는가?
- 이미지 태그(`![...]`)가 각 챕터에 올바르게 매핑되어 있는가?

### 5. 편집 품질 (0~20점)
- 오탈자, 띄어쓰기, 문장 부호 오류가 없는가?
- 마크다운 형식이 올바른가?

## 입력

### 영문 원문:
{original_text}

### 한국어 번역 원고:
{translated_text}
